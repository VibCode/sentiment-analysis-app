# Sentiment Analysis Web App

A beginner-friendly project using Python, Flask, and scikit-learn.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python train_model.py

3. Start the web app:
   python app.py

4. Visit in your browser:
   http://127.0.0.1:5000/
